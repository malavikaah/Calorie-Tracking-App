# Calorie-Tracking-App
🥗 CaloTrack | Cross-platform Flutter nutrition app with hybrid YOLOv8+TFLite on-device food recognition (&lt;2sec, offline) &amp; OpenRouter(GPT-4o/Gemini) cloud fallback. BMI/TDEE profiling, Diabetes &amp; Hypertension alerts, dietitian consultation, PDF reports, Nutrition Encyclopedia, water tracker, CaloCredits &amp; multi-theme UI. Built with Firebase+Dart.
